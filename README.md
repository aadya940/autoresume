# autoResume

**Build professional resumes with AI and automatically find the jobs that match them.**

<img src="app/frontend/public/autoresume-logo.png" width="200" alt="autoResume Logo" />

</div>

## Quick Start

```bash
git clone https://github.com/aadya940/autoresume.git
cd autoresume
docker compose up --build
```

Open [http://localhost:5173](http://localhost:5173) and enter your Gemini API key in the settings icon.

## Features

- **AI Generation**: Turn your profiles into LaTeX resumes
- **Job Match**: Automatically find jobs for your skills
- **Local First**: Your data stays on your machine
- **Live Edit**: Real-time LaTeX preview

## Tech

- **Stack**: React, FastAPI, Docker
- **AI**: Google Gemini
- **Core**: LaTeX, JobSpy
